Sincroniza niveles de membresía de Paid Memberships Pro con productos de WooCommerce para gestionar suscripciones recurrentes.

== Description ==

Este plugin conecta la potencia de Paid Memberships Pro para la gestión de membresías con las capacidades de WooCommerce para el procesamiento de pagos recurrentes. Permite que los niveles de membresía de PMPRO se asocien con productos de suscripción de WooCommerce, automatizando la sincronización del estado de la membresía en función de los pagos y suscripciones de WooCommerce.

Características clave:
* Sincronización automática de estados de membresía.
* Gestión de pagos recurrentes a través de WooCommerce Subscriptions.
* Panel de administración para configuración y logs detallados.

== Installation ==

1. Sube la carpeta `pmpro-woo-sync` al directorio `/wp-content/plugins/`.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Configura las opciones del plugin en 'PMPRO-Woo Sync' -> 'Ajustes' en tu panel de administración de WordPress.
